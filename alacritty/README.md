# Installation

Add this to `~/.config/alacritty/alacritty.yml`

## Forest Night

```yaml
colors:
  # Default colors
  primary:
    background: "0x3c4c55"
    foreground: "0xffebc3"

  # Normal colors
  normal:
    black: "0x7f8f9f"
    red: "0xfd8489"
    green: "0xa9dd9d"
    yellow: "0xf0aa8a"
    blue: "0xbdd0e5"
    magenta: "0xdaccf0"
    cyan: "0xa9dd9d"
    white: "0xffebc3"

  # Bright colors
  bright:
    black: "0x7f8f9f"
    red: "0xfd8489"
    green: "0xa9dd9d"
    yellow: "0xeed094"
    blue: "0xbdd0e5"
    magenta: "0xdaccf0"
    cyan: "0xa9dd9d"
    white: "0xffebc3"
```
